# Zootopia

Zootopia fetches information of animals that fit the users query and creates a website with relevant information on the animals

## Installation

To install this project, simply clone the repository and install the dependencies in requirements.txt using `pip`
You need to register for an api key on https://api-ninjas.com/ and install the key through an .env file.

## Usage

To use this project, run the following command - `python main.py`.
